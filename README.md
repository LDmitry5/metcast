# MetCast - приложение для отображения текущей погоды

Vue 3 приложение для отображения текущей погоды с использованием OpenWeatherMap API.

## Технологии

- Vue 3 (Composition API)
- TypeScript
- Webpack 5
- Axios
- Element Plus

## Установка

1. Установите зависимости:

```bash
npm install
```

2. Создайте файл `.env` в корне проекта и добавьте ваш API ключ от OpenWeatherMap:

VITE_OPENWEATHER_API_KEY=your_api_key_here

Для получения бесплатного API ключа зарегистрируйтесь на [OpenWeatherMap](https://openweathermap.org/api).

## Запуск

### Режим разработки

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:8080`

### Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут находиться в папке `dist/`

## Функционал

- Автоматическое определение местоположения при загрузке
- Отображение текущей погоды (температура, иконка, описание, влажность, скорость ветра, давление)
- Поиск города вручную через поле ввода
- Обработка ошибок (геолокация, API, сетевые ошибки)
- Индикатор загрузки при запросах
- Адаптивная верстка для мобильных устройств и десктопа
